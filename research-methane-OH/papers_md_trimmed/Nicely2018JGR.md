# 1. Introduction

A realistic representation of the atmospheric lifetime of methane $\left( \mathsf { C H } _ { 4 } \right)$ is important for the accurate modeling of global warming. Methane is the second most important anthropogenic greenhouse gas (IPCC,2013). The mixing ratio of tropospheric methane was nearly constant at around 1775 ppb from about 1997 to 2006. Since 2007,methane has been rising at a rate varying between 4.7 ppb/year and 12.5 ppb/year (or $0 . 3 \mathrm { - } 0 . 7 \% /$ year).The cause of the recent rise in methane is debated: some ascribe the rise to emissions from natural wetlands (Schaeferetal,2016),others toacombination of fosil fuelsand wetlands (Kirschke etal.,2013), others to changes in livestock and agriculture (Saunois et al.,2017; Wolf et al.,2017),and stillothers toa decline in the main sink of methane,reaction with the hydroxyl radical (OH)(McNorton et al.,2016; Rigby et al.,2017; Turner et al., 2017).

Our paper is motivated by the temporal variations in the atmospheric loss of methane,which is generally assumed to be constant on the decadal time scale.The loss of methane in the atmosphere is dominated by its reaction with OH in the troposphere (Lelieveld etal.,1998).Due to its role in oxidizing methane and many other atmospheric species,we strive to understand the time-varying evolution of the global burden of tropospheric OH $\left( [ \mathsf { O H } ] ^ { \mathsf { T R O P } } \right)$ .

Numerous modeling and chemical inversion studies,in which observations and known emissons of another species are used to quantify OH concentrations,point to gaps in our knowledge of the value and variability of $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ .For example,calculations conducted using numerous global models showa $6 0 \% 8 0 \%$ variation in the value of the methane lifetime due to loss via OH(Fiore etal.,2009; Nicely etal.,2016,2017;Shindell etal, 2006; Voulgarakis etal.,2013).Nicelyet al.(2016) showed that theobservationally constrained tropospheric column abundance of OH in the Tropical Western Pacific during winter 2014 tended to exceed the value found in global models,due to the tendency of the models to underestimate the observed abundances of ${ { \mathrm O } _ { 3 } }$ and $\mathsf { N O } _ { x }$ $( = { \mathsf { N O } } + { \mathsf { N O } } _ { 2 } )$ .Numerous other studies have also shown that models systematically

Reanalysis fields are used to evaluate the role of ${ \sf H } _ { 2 } \mathrm { O }$ prior to the availabilityof tropospheric observations from satellite (i.e.,before 20o2)as wellas examine the impactof temperature changes on $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ .With the aid of a global chemical transport model, we also include the effect of time varying emissions of ${ \mathsf { N O } } _ { x } \left( = { \mathsf { N O } } + { \mathsf { N O } } _ { 2 } \right)$ on $[ \mathsf { O H } ] ^ { \mathsf { T R O P } }$ .Finally, we investigate the role that the expansion of the tropics (Seidel et al.,2008),a phenomenon not captured in most global models,has in influencing the variabilityof[OH]TROP.

Primary production of OH occurs via ${ { \mathrm O } _ { 3 } }$ photolysis to excited state ${ \mathsf { O } } ( { } ^ { 1 } { \mathsf { D } } )$ followed by reaction with water vapor:

$$
0 _ { 3 } + h \nu ( \lambda \leq 3 3 6 \mathsf { n m } ) \to 0 \big ( ^ { 1 } \mathsf { D } \big ) + 0 _ { 2 } ,
$$

$$
{ \mathsf { O } } { \left( { } ^ { 1 } { \mathsf { D } } \right) } + { \mathsf { H } } _ { 2 } { \mathsf { O } } \to 2 { \mathsf { O } } { \mathsf { H } }
$$

Because this formation pathway depends on the abundance of ${ \sf H } _ { 2 } { \sf O }$ and the actinic flux at wavelengths $\leq 3 3 6 \ \mathsf { n m } _ { \ r { \ r } }$ ，the highest concentrations of OH are found in the tropical lower and middle troposphere. Stratospheric column ${ { \mathrm O } _ { 3 } }$ is low in the tropics,allowing greater penetration of ultraviolet (UV) photons to the troposphere than in the extratropics.This,along with more $H _ { 2 } O$ due to higher temperatures,results in $[ \mathsf { O H } ] ^ { \mathsf { T R O P } }$ being dominated by OH in the tropics (Figure 1).

Secondary sources of OH include recycling of ${ \mathsf { H O } } _ { 2 }$ by reaction with NO:

$$
{ \mathsf { N O } } + { \mathsf { H O } } _ { 2 } \to { \mathsf { N O } } _ { 2 } + { \mathsf { O H } }
$$

Lelieveld etal.(2016)show that on a global basis,this source of OH is close in magnitude to primary production by ${ \mathsf { O } } ( ^ { 1 } { \mathsf { D } } ) + { \mathsf { H } } _ { 2 } { \mathsf { O } }$ $3 3 \%$ of all tropospheric OH production occurs by primary production, while $3 0 \%$ occurs by $N O + H O _ { 2 } )$ . Particularly in strong source regions of $\mathsf { N O } _ { x } , \mathsf { O H }$ abundances are likely determined by this recycling pathway. Additionally,alkyl peroxy radicals $( \mathsf { R O } _ { 2 } )$ that are generated following the oxidation of volatile organic compounds (VOCs) can either react with NO or $\mathsf { H O } _ { 2 } ,$ with pathways leading to the regeneration of OH.To illustrate with the simplest hydrocarbon chemistry,the methane oxidation product methylperoxy radical $( \mathsf { C H } _ { 3 } \mathsf { O } _ { 2 } )$ can react as follows:

$$
N O + C H _ { 3 } O _ { 2 }  N O _ { 2 } + C H _ { 3 } O
$$

$$
\mathsf { C H } _ { 3 } \mathsf { O } + \mathsf { O } _ { 2 } \to \mathsf { H C H O } + \mathsf { H O } _ { 2 }
$$

The ${ \mathsf { H O } } _ { 2 }$ produced in (R5)can then recycle to OH via (R3),while the formaldehyde (HCHO) product can either generate ${ \mathsf { H O } } _ { x } \left( = { \mathsf { O H } } + { \mathsf { H O } } _ { 2 } \right)$ radicals if photolyzed or have a neutral effect following reaction with OH and subsequent production of ${ \mathsf { H O } } _ { 2 }$ Alternatively, ${ \mathsf { C H } } _ { 3 } { \mathsf { O } } _ { 2 }$ can react with ${ \mathsf { H O } } _ { 2 }$ as follows:

$$
H O _ { 2 } + C H _ { 3 } O _ { 2 }  C H _ { 3 } O O H + O _ { 2 }
$$

This reaction pathway is more common in the background atmosphere, where concentrations of $\mathsf { N O } _ { x }$ are lower (Jacob et al.,2005; Muller etal.,2016).Similar to HCHO,the methylhydroperoxide product $( \mathsf { C H } _ { 3 } 0 0 \mathsf { H } )$ either produces $\mathsf { H O } _ { x }$ or has no influence on $\mathsf { H O } _ { x } ,$ depending on whether loss of ${ \mathsf { C H } } _ { 3 } { \mathsf { O O H } }$ occurs by photolysis or by reaction with OH.

Loss of $\mathsf { H O } _ { x }$ radicals is controlled mainly by the self-reaction of $\mathsf { H O } _ { 2 } ,$ (R7), especially in the low $\cdot \mathsf { N O } _ { x }$ environment that dominates the tropics where the bulk of the tropospheric OH abundance exists.

$$
\mathsf { H O } _ { 2 } + \mathsf { H O } _ { 2 } \to \mathsf { H } _ { 2 } \mathsf { O } _ { 2 } + \mathsf { O } _ { 2 }
$$

The hydrogen peroxide $( \mathsf { H } _ { 2 } \mathsf { O } _ { 2 } )$ product is highly soluble and readily removed from the atmosphere through wet deposition.In areas with elevated $\mathsf { N O } _ { x } ,$ OH can also be removed through reaction with ${ \mathsf { N O } } _ { 2 }$ to form another soluble product, nitric acid $( \mathsf { H N O } _ { 3 } )$ ：

$$
\mathsf { O H } + \mathsf { N O } _ { 2 } + \mathsf { M } \to \mathsf { H N O } _ { 3 } + \mathsf { M }
$$

However,relatively low- $\cdot \mathsf { N O } _ { x }$ conditions prevail throughout the bulk of the troposphere,meaning that the self-reaction of ${ \mathsf { H O } } _ { 2 }$ (R7) is the dominant sink for $\mathsf { H O } _ { x }$ radicals (Jaeglé et al., 2001).Throughout this study, we consider the chemistry of $\mathtt { O H }$ in the context of the $\mathsf { H O } _ { x }$ family due to our interest in the mean oxidation state of the troposphere and how it evolves over time.If one were interested in the fast chemistry of the OH radical over short time scales,a strict budgeting of OH loss by its reactions with a host of VOC species and its recycling via a wide range of ${ \mathsf { H O } } _ { 2 }$ and ${ \mathsf { R O } } _ { 2 }$ reactions would be required. Considering the $\mathsf { H O } _ { x }$ radical group as a whole,though,allows us to simplify the key chemistry to its primary production (R1)followed by(R2) and loss pathways (R7) and (R8)).

While the chemistry detailed here illustrates the simplest secondary production pathways of OH, Lelieveld et al. (2016) posit the importance of recycling OH from more complex oxygenated VOCs (OvOCs) in continental, low $\mathsf { N O } _ { x }$ regimes,such as the Amazon basin.The chemistry of OvOCs is represented in our baseline model,and the effect of OVOCs on OH described by Lelieveld et al. (2016) is expected to be limited to a very small altitude range (i.e., $1 { - } 2 \ \mathsf { k m } )$ ）over tropical forests,because of the short lifetime of the compounds that drive this chemistry. Due to the lack of observational constraints on temporal variations of OVOCs (Kim et al.,2015; Read et al.，2012) and knowledge gaps concerning the recycling pathway of OVOCs (Brewer etal,2017; Milet etal.,2010; Muler etal.,2016; Naik etal.,2010),we have not included this term in our assessment of factors that govern changes in the oxidizing capacity of the global troposphere.

Evidence from the past two decades,based on various diagnostics of tropical extent,suggests the climatological tropics are expanding.This phenomenon can also be viewed as a strengthening or widening of the Hadley circulation,since the location of descent is generally regarded as the boundary between tropics and extratropics.Numerous studies estimating the rate of tropical widening are summarized by Seidel et al.(2008).They include rates of widening determined by analyzing tropopause heights (Birner,2010; Lu et al.,2009;Seidel&Randel,2007),subtropical jetstreamlocations (Archer&Caldeira,2008;Fu etal2006; Fu& Lin,2011; Hu&Fu,2007),meridional mass streamfunctions (Hu&Fu,2007; Mitas &Clement,2005),outgoing longwave radiation (Hu & Fu, 20o7),and total ${ { \mathrm O } _ { 3 } }$ columns (Hudson,2012; Hudson et al., 2006; Follette-Cook et al., 2009).

The findings of observational studies of tropical widening are often presented with inconsistent context, fol example,reporting rates of widening representative ofone hemisphere versus both hemispheres.To simplify interpretation of theliterature,we summarize early analyses estimated rates of expansionof the tropical belt, globally,as high as $3 . 0 ^ { \circ } ,$ /decade (Seidel& Randel,2oo7); however,refinement of methods and continued observation have resulted in lower estimates (Allen et al.,2014; Birner, ${ 2 0 1 0 } ;$ Davis & Rosenlof,2012).We emphasize that allof these studies are based on observations or reanalysis data sets.The most comprehensive analyses to date are Davis and Rosenlof (2012),who reported that the meridional mass streamfunction shows a robust $1 . 0 ^ { \circ }$ to $1 . 5 ^ { \circ } { } ^ { } ,$ /decade global rate of widening,while other diagnostics suggest insignificant trends,and Allen et al. (2014),who calculated rates of $0 . 3 5 \pm 0 . 0 9 ^ { \circ } ,$ /decade in the Northern Hemisphere (NH) and $0 . 1 7 \pm 0 . 1 0 ^ { \circ } /$ decade in the Southern Hemisphere (SH),as well as $0 . 5 2 \pm 0 . 1 3 ^ { \circ } /$ decade globally (all based on observations).Despite the latest evidence that these rates are lower than first suggested,concerns persist due to the inability of global models to simulate the extent of tropical widening witnessed to date (Allen et al.,2014; Johanson & Fu,2009).In general, while many models do simulate a widening of the Hadley cellin response to climate change,nearly all models considerably underestimate the magnitude of the observed widening (Allen et al., 2014). Here we quantify the effect of tropical widening on $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ which has yet to be considered in the literature.

The mechanism through which widening of the tropics ikely afects OH chemistry is through the exposure of a larger portion of the troposphere to regions with low amounts of overhead column ${ { \mathrm O } _ { 3 } }$ .Total column ${ { \mathrm O } _ { 3 } }$ is lowest in the tropics due to the upward motion of the Brewer-Dobson overturning circulation (Seidel etal, 2008).This large-scale flow pattern carries stratospheric ${ { \mathrm O } _ { 3 } }$ from the tropics poleward, resulting in a buildup of $\mathsf { O } _ { 3 } ,$ and hence higher columns,in the extratropics.Indeed,the boundary between high and low column $O _ { 3 }$ air masses can be used to distinguish the extratropics from the tropics (Hudson et al.,2006).As with the Hadley circulation in the troposphere, modeling studies find that increasing greenhouse gas concentrations willresult in strengtheningof theBrewer-Dobson circulation (Butchart etal.,2006; Garcia & Randel,2008;Li et al.2010),so it logically follows that the region of low column ${ { \mathrm O } _ { 3 } }$ will expand in conjunction with the climatological tropics.This phenomenon has been observed in measurements of total column ${ { \mathrm O } _ { 3 } }$ (Hudson, 2012; Hudson etal.,206).Intheregions that transition fromextratropical to tropicalclassification,lower column ${ { \mathrm O } _ { 3 } }$ willincrease UV photon flux into the troposphere,likely contributing to higher vaues of $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ and thus reduced lifetime of methane. Further discussion of the mechanisms that we hypothesize are driving changes in $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ is provided in section 2.5.6.

In this study,weempirically modeltheanges in $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ that occurred from 1980-2015 by considering global observations of overhead ${ \sf O } _ { 3 } , { \sf H } _ { 2 } { \sf O } ,$ methane,and temperature as well as the simulated effects of varying $\mathsf { N O } _ { x }$ emisions and tropical expansion.The effect of each component on $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ is evaluated based on fundamental principles of atmospheric chemistry.Results of this analysis quantify which contributors to variations in $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ are most importanttoaccurately model past and future long-term trends in the oxidizing capacity of the troposphere.

# 2. Methods

# 2.1.GMI Model

We use a 3-Dfield of OH calculated bythe Global Modeling Initiative(GMl)chemical transport model(Duncan etal.,2007;traaetal,asteartigsenaroforthedistributioanddenf $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ . Output foryear 1980 is taken from alonger historical simulation conducted by GMlfor the Chemistry-Climate Model Initiative (CCMl; Eyringetal.,2013; Morgensternetal.,2017).Specificaly,thisoutput is taken from the Ref-C1SD historical specified dynamics simulation covering years 1980-2016. Monthly mean OH concentrations are output on a $2 . 5 ^ { \circ }$ longitude $\times 2 ^ { \circ }$ latitude grid,with 72 vertical layers from the surface to $0 . 0 1 5 { \mathsf { h P a } } .$ While it may be desirable to initialize our OHfield with a comprehensive setof box model simulations constrained by observations,the many species and parameters that influence OH chemistryare not measured suffciently at the global scale to enable such a determination of $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ .Instead,we relyon the GMl chemical transport model,which in turn utilizes realistic meteorological parameters and emissions of trace gases.GMl is constrained to meteorological fields from the Modern-Era Retrospective Analysis for Researchand Applications Version 2(MERRA-2) reanalysis data set (Gelaro et al.,2017)and run with a tropospherestratosphere chemical mechanism that includes 117 species,322 thermal reactions,and 81 photolysis reactions (Duncan et al.,20o7; Strahan etal.,2007).For notes on availabilityof thisandallforthcoming data sets, see the Acknowledgments section.

Anthropogenic and biomass burning emissions for the CCMl simulation come from the MACCity inventory (Granier et al.,2011).The MACCity inventory linearly interpolates the decadal Atmospheric Chemistry and Climate Model Intercomparison Project emissions (Lamarque et al.,2010) to each year on a sector-by-sector basis until 2000. MACCity anthropogenic emissons for 2000 to 2010 are based on the Representative Concentration Pathway 8.5.Biomass burning emissions are based on a combination of the Reanalysis of the Tropospheric chemical composition over the past 40 years (RETRO)inventory(Schultz etal.,2008),which spans the period 1960-2000,and the Global Fire Emissions Database version 2(GFED2) (van der Werf et al, 2006)for 1997-2008,with the two data sets harmonized to avoid an unphysical jump (Lamarque etal.,2010). While Lamarque et al. (2010)only provide decadal monthly mean biomass burning emissions, resulting in a lack of interannual variability,Granier et al. (2011) calculate monthly emissions on an annual basis for the MACCity inventory using the RETRO and GFED2 carbon emisson data sets.The GFED2 inventory uses burned area and fire count data from the Moderate-Resolution Imaging Spectroradiometer (MODIS)instrument on board the Terra satelite to generate biomass burning emissions.As such,a model constrained to these emissions could be expected to reasonably reproduce variations in $\mathsf { N O } _ { x }$ that have occurred due to biomass burning. Lightning $\mathsf { N O } _ { x }$ $( \mathsf { L N O } _ { x } )$ is calculated online within GMl following Allen et al. (2010) and varies intraannually, though annual emissons are constrained to 6.5Tg N/year.GMl also includes an online calculation of $\mathsf { N O } _ { x }$ emissions from soil,which vary depending on temperature and precipitation following Yienger and Levy I (1995).

# 2.2.DSMACC Box Model

# 2.3. Calculation of [OH]TROP

The calculation of $[ \mathsf { O H } ] ^ { \mathsf { T R O P } }$ requires amassweighting toaccountfor thediminishing area of higher-latitude bands and the decreasing density of air at higher altitudes.Inaddition,areactivity weighting is used to find a value of [OH]TROP appropriate for the oxidation of methane.We use the methane mass and reaction rate weightings that were suggested by Lawrence et al. (2001):

$$
[ 0 \mathsf { H } ] ^ { \mathsf { T R O P } } = \frac { \Sigma ( [ 0 \mathsf { H } ] \times k _ { 0 \mathsf { H } + \mathsf { C } \mathsf { H } 4 } ( T ) \times M _ { \mathsf { C } \mathsf { H } 4 } ) } { \Sigma ( k _ { 0 \mathsf { H } + \mathsf { C } \mathsf { H } 4 } ( T ) \times M _ { \mathsf { C } \mathsf { H } 4 } ) }
$$

Here [OH], $T ,$ and $M _ { \mathrm { C H 4 } }$ are monthly mean values of OH number density,temperature,and mass of methane in each GMl model grid box, respectively.Summations cover allatitudes,longitudes,and pressure levels within the troposphere.Tropopause altitudes are standard model output,based on a blended definition that includes vertical thermal gradients and potential vorticity (Wilcox et al., 2012).

# 2.5. Calculation of the OH Response

We start with the base 3-D field of OH concentrations obtained from the GMl model.The value of $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ calculated from this field for each month serves as our initial burden,which we assume to be representative of 1980.We then calculate perturbations to this OH field using OH sensitivities found with the DSMACC box model and observed changes in the sensitivity parameters.For example,combining the OH sensitivity to methane with the observed change in methane yields the change in OH due to methane. Inother words,the box model-calculated sensitivity is multiplied ofine by the change in the variable being examined.The perturbed OH field is then used to recalculate $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ for all years of our analysis. The difference between the perturbed $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ and the baseline 1980 value then represents the impact on $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ due to the individual factors (overhead ${ \sf O } _ { 3 } , { \sf H } _ { 2 } { \sf O } ,$ methane, $\mathsf { N O } _ { x } ,$ and tropical widening) we examine. Below,we describe the relationships between OH and these variables used to calculate the $[ \mathsf { O H } ] ^ { \mathsf { T R O P } }$ response.

# 2.5.1. OH Response to Overhead $\pmb { 0 _ { 3 } }$

The process we use to evaluate the response f $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ to variations in overhead ${ { \mathrm O } _ { 3 } }$ involves several distinct steps. Observations of total column ${ { \mathrm O } _ { 3 } }$ from the MOD data set described in section 2.4 are used to derive observation-based variations in J $( { \mathsf { O } } ^ { 1 } { \mathsf { D } } )$ .Box model calculations are also used to find the sensitivity of [OH] to changes in $[ 0 ^ { 1 } \mathsf { D } ]$ .We evaluate this sensitivity because it is the direct mechanism through which OH primary production is altered. First, we provide details on how the calculation of J $( 0 ^ { 1 } \mathsf { D } )$ using observations of total column ${ { \mathrm O } _ { 3 } }$ is carried out.

Output from the box model simulations are used to calculate δ [OH]/8 $[ 0 ^ { 1 } \mathsf { D } ]$ as a function of local $[ { \mathrm { O } } _ { 3 } ] ,$ overhead $\mathrm { O } _ { 3 } ,$ latitude,and pressure.We use ${ C _ { \mathrm { O 1 D } } }$ to denote δ $[ \mathsf { O H } ] / \delta [ \mathsf { O } ^ { 1 } \mathsf { D } ]$ .The units of $\zeta _ { \mathsf { O } 1 \mathsf { D } }$ are percecnt change of[OH] divided by percent change in $[ 0 ^ { 1 } \mathsf { D } ]$ . # 2.5.2. OH Response to ${ \sf H } _ { 2 } { \sf O }$

The sensitivity of [OH] to variations in $[ \mathsf { H } _ { 2 } \mathsf { O } ]$ is also evaluated using the DSMACC box model.Calculation of $C _ { H 2 0 }$ is performed as described above except as a function of varying ${ \sf H } _ { 2 } { \sf O }$ and $\mathsf { N O } _ { x } .$ The sensitivity is interrogated as a function of $\mathsf { N O } _ { x }$ because the extent to which OH production occurs via primary (R1) and (R2)) versus secondary (R3) routes determines how strongly [OH] responds to variations in $[ \mathsf { H } _ { 2 } \mathsf { O } ]$

The $[ \mathsf { O H } ] ^ { \mathsf { T R O P } }$ response to ${ \sf H } _ { 2 } { \sf O }$ is found by calculating the relative change in ${ \sf H } _ { 2 } { \sf O }$ on a monthly basis from both MERRA-2 and AIRS. # 2.5.3. OH Response to Methane

The effectof methane on $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ is evaluated by using the DSMACC box model to estimate the feedback of methane on OH,denoted as ${ \cal C } _ { \mathsf { C H } 4 } ,$ at varying levels of $\mathsf { N O } _ { x }$ .Within each of the six latitude bins and four pressure bins identified in section 2.2,all tropospheric values of $\mathsf { N O } _ { x }$ present in the GMl simulation are used to identify $1 0 ~ \mathsf { N O } _ { x }$ bins.Mixing ratios of methane are varied in 10 steps spanning the observational record for our study period (1,640-1,850 ppb,determined from Dlugokencky et al.,2009).Allother inputs to the box model are determined as averages of tropospheric values from GMl within each latitude and pressure bin. Output from these simulations are used to calculate $C _ { \mathsf { C H 4 } }$ as a function of methane and $\mathsf { N O } _ { x } ,$ in units of percent change in [OH] per parts per bilion change in methane.Each value of OH in the 3-D base field is then adjusted according to the value of $C _ { \mathsf { C H 4 } }$ for the appropriate latitude, pressure,and $\mathsf { N O } _ { x }$ bin. # 2.5.4. OH Response to Temperature

The effect of temperature variations on $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ is calculated to take into account both changes in chemical production of OH as wellas its rate of reaction with methane.We perform box model calculations of S (OH)/8 (T) as a function of $[ 0 _ { 3 } ]$ following the same method of our methane sensitivity calculation,that is, probing conditions that span the full range of temperature and $[ 0 _ { 3 } ]$ within the troposphere.This sensitivity is applied within the calculation of $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ . Note that since the box model simulations are constrained by values of $[ \mathsf { H } _ { 2 } \mathsf { O } ]$ determined for the latitude and pressure bins identified,the effect of temperature on $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ via water vapor is not evaluated here.Instead,that effect is encompassed by our evaluation of ${ \sf H } _ { 2 } { \sf O }$ described in section 2.5.2. Simultaneously，temperature values used in the calculation of the rate constant for the $\mathsf { O H } + \mathsf { C H } _ { 4 }$ reaction, or $k _ { \mathrm { O H } + \mathrm { C H } 4 }$ in equation (1),are allowed to vary based on year. As such, the total response of $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ to temperature encapsulates both changes in OH production and loss.

# 2.5.6. OH Response to Tropical Expansion

To simulate the change in $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ that resulted from expansion of the tropics over the period of 1980-2015,we first have to determine the extent of the tropics in our initial 3-D feld of OH from the GMl model. We use the bimodal distribution of tropopause heights, first discussed in

Seideland Randel (2007),to determine Hadleycellboundaries.While Seideland Randel (2007) suggestadefinition of tropical extent dependent on an absolute threshold in tropopause altitude, Davis and Rosenlof (2012) offer a more robust definition that instead uses relative changes in tropopause altitude.Following the Davis and Rosenlof (2012) definition, we identify the tropical boundaryas the lowest latitude at which the tropopause altitude falls to $1 . 5 ~ \mathsf { k m }$ below the mean tropopause altitude between $1 5 \%$ and $1 5 \mathrm { { } ^ { \circ } N . }$ We tested everal additionaldefinitions of tropical extentbut found thataveraging over the relatively short time span of 1 model year (from 12 monthly averages) was insuficient for producing smooth,unbroken tropical boundaries for all but the tropopause altitude metric. We determine tropical boundaries using tropopause height for each longitude,in both the NHand SH,for each month of the GMl base simulation.The tropical boundaries that we identify are shown as black lines in Figure 1.

The primary mechanism through which we expect tropical widening to affect $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ is primarily through the change in overhead $\mathsf { O } _ { 3 } ,$ which in turn follows variations in the tropopause height (Hudson et al., 2003). Increases in ${ \sf H } _ { 2 } { \sf O }$ that would accompany the transition of a location from an extratropical to a tropical regime would also tend to increase OH concentrations.The corelation of OH with tropopause height is clear when analyzing zonal mean values of tropospheric OH column as output by the GMl model (Figure 2).The values depicted in Figure 2 are also annually averaged for ilustration purposes,though we note that analysis of simulated tropical widening is performed ona monthly basis.The Pearson correlation coeficient of annually, zonally averaged OHcolumn versus tropopause altitude is 0.99.Without averaging,the correlation coeficient of the longitude dependent, monthly values is 0.85 due to zonal asymmetries in the OHcolumn that are driven by factors other than tropopause altitude and overhead ${ { \mathrm O } _ { 3 } }$ (such as $\mathsf { N O } _ { x } )$ .We impose the effects of tropical expansion by essentially widening the latitudinal distribution of tropospheric OH column values,seen as the black doted line in Figures 2and S8,which shows azoomed-in view of the NH.This is doneby placing anchor points at the minimum and maximum latitudes identified as tropical boundaries,as a function of longitude,fora given hemisphere and month.The OH column value atthe mean tropical boundary latitude is shifted poleward by the number of degrees of tropical widening being imposed. Here we use the rates of $0 . 3 5 ^ { \circ } { } ^ { } ,$ /decade in the NH and $0 . 1 7 ^ { \circ }$ decade-1 in the SH as estimated from observations by Allen et al.(2014) to simulate therateof Hadleycellexpansionandapproximate theresulting sensitivityof $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ . We also use values of $0 . 5 ^ { \circ }$ decade-1 in each hemisphere to represent the maximum rate,reported by Davis and Rosenlof (2012)through examination of mean meridional streamfunction,and of $0 . 0 5 ^ { \circ }$ decade-1 in the NH,

$0 . 1 3 ^ { \circ }$ decade-1 in the SH to represent the rate of widening simulated by models that participated in the Coupled Model Intercomparison Project Phase 5(CMIP5)(Alen et al.,2014).The poleward shift of tropospheric OH columns maximizes at the mean tropical boundary latitude and is interpolated to a shift of $0 ^ { \circ }$ at the anchor points.We calculate the fractional change in the tropospheric OH column at each latitude, depicted in Figure S9,using this new distribution (black dashed line,Figures 2and S8),andapplythis within our calculation of[OH]TROP

We consider the efects of tropical widening and changes in overhead column ${ { \mathrm O } _ { 3 } }$ on $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ as independent factors in this analysis.The justification for doing so is that the primary effect of tropical widening on $[ \mathsf { O H } ] ^ { \mathsf { T R O P } }$ is driven by theexposure of more regions of the troposphere to climatological tropical conditions (low column $\mathsf { O } _ { 3 } ,$ high humidity), rather than a variation of either column ${ { \mathrm O } _ { 3 } }$ or humidity within the latitudes defined to be the tropics at any point in time.In other words,the third and fourth latitude bins $( 3 0 ^ { \circ } \mathsf { S } - 0 ^ { \circ } ;$ $0 ^ { \circ } - 3 0 ^ { \circ } \mathsf { N }$ section 2.2) used in our analysis should be expanded to account for the widening of the tropics, whereas the second and fifth bins $( 6 0 ^ { \circ } \mathsf { S } _ { } { - } 3 0 ^ { \circ } \mathsf { S } ;$ $3 0 ^ { \circ } \mathsf { N } \mathrm { - } 6 0 ^ { \circ } \mathsf { N } )$ should be contracted to compensate. Furthermore,the observed changes in total ${ { \mathrm O } _ { 3 } }$ result from factors not related to tropical expansion, including the decreasing atmospheric burden of halogenated compounds (Butchart & Scaife,2001; Li etal.,2009)and the injectionof volcanic sulfate aerosols into the stratosphere (Fahey et al.,1993),both of which impart larger changes in overhead ${ { \mathrm O } _ { 3 } }$ than are expected as a result of tropical expansion.Additionally, increases in ${ \sf H } _ { 2 } { \sf O }$ are expected to accompany the decreases in overhead ${ { \mathrm O } _ { 3 } }$ as a location transitions to a tropical regime.Given the multiple factors thatlikelyaffect $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ during theprocessoftropical widening and the lowlevelof agreement between metrics of Hadley cell expansion (Davis & Rosenlof,2012),we choose to approximate the effect by estimating the impact on $\mathsf { [ O H ] } ^ { \mathsf { T R O P } }$ of changing the boundaries ofour model boxes,albeit in simplified fashion.It is possble that other complex processes,such as an increasing magnitude of stratospheretroposphere ozone flux as a result of climate change (Hegglin & Shepherd,2009),also influence $[ { \mathsf { O H } } ] ^ { \mathsf { T R O P } }$ though we have notsought to examine those here.Ultimately,improving the dynamics within global models such that tropical expansion and other changes in circulation are simulated with accuracy willenable a more process-oriented approach to this question.
