Abstract. The methane chemical sink estimated by atmospheric chemistry models (bottom-up method) is significantly larger than estimates based on methyl chloroform (MCF) inversions (top-down method). The difference is partly attributable to large uncertainties in hydroxyl radical (OH) concentrations simulated by the atmospheric chemistry models used to derive the bottom-up estimates. In this study, we propose a new approach based on OH precursor observations and a chemical box model. This approach contributes to improving the 3D distributions of tropospheric OH radicals obtained from atmospheric chemistry models and reconciling bottom-up and top-down estimates of the chemical loss of atmospheric methane. By constraining simulated OH precursors with observations, the global mean tropospheric column-averaged air-mass-weighted OH concentration $( \mathrm { [ O H ] _ { t r o p - M } } )$ is $\sim 1 0 \times 1 0 ^ { 5 } \mathrm { \ m o l e c . } \mathrm { c m } ^ { - 3 }$ (which is $2 \times 1 0 ^ { 5 } \mathrm { m o l e c . c i m } ^ { - 3 }$ lower than the original modelsimulated global $[ \mathrm { O H } ] _ { \mathrm { t r o p - M } } )$ and agrees with that obtained by the top-down method based on MCF inversions. With OH constrained by precursor observations, the methane chemical loss is $4 7 1 { - } 5 0 8 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ , averaged from 2000 to 2009. The new adjusted estimate is in the range of the latest top-down estimate of the Global Carbon Project (GCP) $( 4 5 9 - 5 1 6 \mathrm { T g } \mathrm { y r } ^ { - 1 } ,$ ), contrary to the bottom-up estimates that use the original model-simulated OH fields $( 5 7 7 - 6 1 2 \mathrm { T g } \mathrm { y r } ^ { - 1 } ,$ ). The overestimation of global $\mathrm { [ O H ] _ { t r o p - M } }$ and methane chemical loss simulated by the atmospheric chemistry models is caused primarily by the models’ underestimation of carbon monoxide and total ozone column, and overestimation of nitrogen dioxide. Our results highlight that constraining the modelsimulated OH fields with available OH precursor observations can help improve bottom-up estimates of the global methane sink.

# 1 Introduction

Methane $\mathrm { ( C H _ { 4 } ) }$ is a potent greenhouse gas, with its 100-year global warming potential of 27 (for non-fossil $\mathrm { C H } _ { 4 }$ ) and 30 (for fossil $\mathrm { C H } _ { 4 }$ ) times that of $\mathrm { C O } _ { 2 }$ (Forster et al., 2021). The tropospheric $\mathrm { C H } _ { 4 }$ mixing ratios have increased by more than 1.6 times between the pre-industrial age and the present day, resulting in $0 . 5 4 { \pm } 0 . 1 1 \mathrm { { W } } { \mathrm { m } } ^ { - 2 }$ of radiative forcing from 1750 to 2019 (Forster et al., 2021). After a short-term stabilization during 2000–2006, the atmospheric methane mixing ratio rose increasingly quickly from $5 \mathrm { p p b v y r } ^ { - 1 }$ in 2006 to $1 7 \mathrm { p p b v y r } ^ { - 1 }$ in 2021, based on surface networks (Dlugokencky, 2022). The rapid growth in atmospheric $\mathrm { C H } _ { 4 }$ over the recent decade further challenges meeting the $1 . 5 – 2 . 0 ^ { \circ } \mathrm { C }$ targets of the Paris Agreement (Nisbet et al., 2019) and is therefore becoming an increasing concern (Jackson et al., 2020).

Understanding the drivers of atmospheric methane changes rely on accurate estimates of the global methane budget, as methane concentrations in the atmosphere are the net balance between emissions and sinks. To estimate this budget, the Global Carbon Project (GCP) has established the global $\mathrm { C H } _ { 4 }$ budget by gathering up-to-date observations and model information (Kirschke et al., 2013; Saunois et al., 2016, 2017, 2020). One of the remaining largest uncertainties, as pointed out by the most recent $\mathrm { C H } _ { 4 }$ budget (Saunois et al., 2020), is the chemical loss of $\mathrm { C H } _ { 4 }$ . The chemical loss of $\mathrm { C H } _ { 4 }$ stems mainly through the reaction of $\mathrm { C H } _ { 4 }$ with hydroxyl radical (OH), which is also the most important $\mathrm { C H } _ { 4 }$ sink.

The hydroxyl radical (OH) is a key species in tropospheric chemistry that reacts with most greenhouse gases and air pollutants (Levy, 1971), being the main oxidant of the lower atmosphere. Due to its extremely short lifetime (typically 1 s) and spatial variability, direct observations do not allow for the quantification of the global OH distributions. The OH for calculating the chemical sink of $\mathrm { C H } _ { 4 }$ is thus estimated either from top-down or bottom-up methods. The top-down method estimates OH mainly through inversions constrained by independent observations of 1-1-1trichloroethane (methyl chloroform, MCF), assuming that emissions of this compound are well known. Such MCF-based top-down methods have been widely used in the scientific community to derive OH trends, but it can only yield the global to latitudinal mean OH due to the sparse MCF observations and does not represent the chemical feedback on OH (e.g., Prinn et al., 2001; Bousquet et al., 2005; Montzka et al., 2011; Naus et al., 2021; Patra et al., 2021). Bottom-up approaches on the other hand simulate the OH by atmospheric chemistry models to account for the chemical mechanisms that determine OH production and loss, but their estimates of the global mean OH usually disagree with MCF-based estimates (Naik et al., 2013; Zhao et al., 2019).

The global OH estimated by bottom-up model-based and top-down MCF-based methods are different in magnitudes, interannual variations, and trends, resulting in large differences in estimated $\mathrm { C H } _ { 4 }$ sinks between the two methods. In the last global $\mathrm { C H } _ { 4 }$ budget, most of the OH fields used to estimate the bottom-up $\mathrm { C H } _ { 4 }$ sink were obtained from the atmospheric chemistry models that participated in the International Global Atmospheric Chemistry (IGAC)/Stratosphere–troposphere Processes and their Role in Climate (SPARC) Chemistry–Climate Model Initiative Phase-1 (CCMI-1) project. However, these models showed a wide range of $9 . { \overset { - } { 4 } } - 1 4 . 4 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ in mean tropospheric column-averaged air-mass-weighted OH $( [ \mathrm { O H } ] _ { \mathrm { t r o p - M } } )$ (Zhao et al., 2019; Saunois et al., 2020), thus mostly higher than the values estimated by the MCF-based inversions $( \sim 1 0 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ ; Prinn et al., 2001; Bousquet et al., 2005). Indeed, the mean $\mathrm { C H } _ { 4 }$ chemical loss for 2000–2009, as calculated by bottom-up approaches, is $5 9 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ (range $4 8 9 { - } 7 4 9 \mathrm { T g } \mathrm { y r } ^ { - 1 } .$ ), much higher than the $5 0 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ (range $4 5 9 { - } 5 1 6 \mathrm { T g } \mathrm { y r } ^ { - 1 } .$ ) estimated by top-down $\mathrm { C H } _ { 4 }$ inversions (Saunois et al., 2020). Those top-down inversions using box models indicate that the OH changes may have influenced recent $\mathrm { C H } _ { 4 }$ trends, although with large uncertainties (Turner et al., 2017; Rigby et al., 2017), while more recent 3D inversions show no significant trend in OH after 2000 (Naus et al., 2021; Patra et al., 2021). In contrast to top-down MCF-based inversions, atmospheric chemistry models simulate a positive decadal trend in OH and consequently $\mathrm { C H } _ { 4 }$ chemical loss from the 1980s (Zhao et al., 2020b).

Reconciling the bottom-up and top-down estimates of the methane chemical sink is essential for a more accurate estimate of the global methane budget and to better attribute the observed changes in atmospheric growth rates of $\mathrm { C H } _ { 4 }$ . One way to improve the bottom-up estimates of the $\mathrm { C H } _ { 4 }$ sink and thus reconcile the difference is to correct the OH simulated by atmospheric models using observations of OH precursors. Indeed, uncertainties in the OH simulated by atmospheric models can be attributed to biases in precursor concentrations. For example, Naik et al. (2013) found that an underestimation of carbon monoxide (CO) in the Northern Hemisphere can contribute to the overestimation of OH in this hemisphere; Strode et al. (2015) estimated that removing the model bias in ${ { \mathrm O } _ { 3 } }$ and water vapor $( \mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g } ) } )$ and reducing northern hemispheric nitrogen oxide $( \mathbf { N O } _ { x } = \mathbf { N O } + \mathbf { N O } _ { 2 } )$ emissions can reduce a high bias in the global tropospheric OH burden by $10 \%$ ; Nicely et al. (2017, 2020) found that the inter-model difference in tropospheric OH is mainly driven by the difference in model-simulated ultraviolet light flux to the troposphere, the tropospheric ${ { \mathrm O } _ { 3 } }$ , CO, and $\mathrm { N O } _ { x }$ mixing ratio. In addition, the budget analysis of OH production and loss showed that about $90 \%$ of OH production is directly related to stratospheric and tropospheric ozone $\left( \mathbf { O } _ { 3 } \right)$ $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ , and nitrogen oxide (NO), and $\sim 6 0 \%$ of OH is removed by reaction with CO, $\mathrm { C H } _ { 4 }$ , and formaldehyde $\mathrm { ( C H } _ { 2 } \mathrm { O ) }$ (Lelieveld et al., 2016; Zhao et al., 2020b). Thus, bottom-up estimates of the $\mathrm { C H } _ { 4 }$ sink from chemistry transport models (CTMs) can be improved if one can reduce the biases due to these OH precursors in modeled OH.

Fortunately, several satellites have collected long-term continuous observations of the aforementioned OH precursors with global coverage, providing the opportunity to evaluate and improve bottom-up estimates of the $\mathrm { C H } _ { 4 }$ sink. The chemistry reanalysis that assimilates satellite observations of ${ { \mathrm O } _ { 3 } }$ , CO, $\Nu { 0 } _ { 2 }$ , nitric acid $\mathrm { ( H N O } _ { 3 } )$ ), and CO shows significant improvement on both global OH burden and interhemispheric gradient (Miyazaki et al., 2020). Such dataassimilation methods can well balance the model and observation uncertainties, but they are not easy to apply to different models that simulate the broad range of global OH burden (Naik et al., 2013; Zhao et al., 2019). In addition, they do not allow partitioning the OH bias due to each precursor. In this context, the main objective of this study is to explore a simple approach to reconcile bottom-up and top-down estimates of the $\mathrm { C H } _ { 4 }$ sink by (i) improving the simulated atmospheric OH fields using multiple satellite observations and meteorological data from reanalysis and (ii) assessing the contribution of each main OH precursor to the bias in simulated OH and $\mathrm { C H } _ { 4 }$ sink. As a result, top-down estimates of $\mathrm { C H } _ { 4 }$ emissions will also benefit from the improved 3D distributions of OH (Zhao et al., 2020a; Saunois et al., 2020). We first evaluate the OH precursors (CO, $\mathrm { C H } _ { 4 }$ , ${ { \mathrm O } _ { 3 } }$ , $\mathrm { C H } _ { 2 } \mathrm { O }$ , and $\Nu { 0 } _ { 2 }$ , the total column ${ { \mathrm O } _ { 3 } }$ , and $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ ) simulated for the year 2010 by the CESM1–CAM4Chem (Community Earth System Model using the Community Atmosphere Model version 4 as atmosphere component) and GEOSCCM (Goddard Earth Observing System Chemistry–Climate Model); these models participated in the CCMI-1 project, and they were used to estimate the global methane sink in Saunois et al. (2020) and represent two different chemical mechanisms. We then estimate the observation-based OH fields by correcting model biases of the two modeled OH fields due to the abovementioned OH precursors using the Dynamically Simple Model of Atmospheric Chemical Complexity (DSMACC). By doing so, we quantify the bias in tropospheric OH attributable to each precursor. Finally, we estimate the chemical sink of $\mathrm { C H } _ { 4 }$ using the observation-based OH field and, based on the uncertainties inferred for OH, we reveal the dominant factors contributing to the uncertainties in $\mathrm { C H } _ { 4 }$ chemical sink at the global and regional scales.

# 2 Method

# 2.1 Observational data

The total column ${ { \mathrm O } _ { 3 } }$ , which mainly influences the O $( ^ { 1 } \mathbf { D } )$ photolysis rate, is constrained by the National Aeronautics and Space Administration (NASA) Solar Backscatter Ultraviolet (SBUV) Merged Ozone Data Set (MOD) (Frith et al., 2014). The SBUV MOD column ${ { \mathrm O } _ { 3 } }$ data are derived by combining observations from nine SBUV-type instruments aboard NASA’s Aura satellite. The monthly ${ { \mathrm O } _ { 3 } }$ columns are available for $5 ^ { \circ }$ zonal mean.

Tropospheric ${ { \mathrm O } _ { 3 } }$ is important in determining OH production. We constrain its spatial distributions using the tropospheric column ${ { \mathrm O } _ { 3 } }$ data from the combined Aura Ozone Monitoring Instrument/Microwave Limb Sounder (OMI/MLS) satellite observations, which are generated by subtracting the co-located MLS limb measurements (integrated over the stratosphere to derive stratospheric column ozone) from total column ozone retrieved by OMI, a UV–Vis nadir solar backscatter spectrometer (Ziemke et al., 2006).

The tropospheric nitrogen oxide family $( \mathrm { N O } _ { x } = \mathrm { N O } +$ $\mathrm { N O } _ { 2 }$ ) participates in both OH production – reaction of nitrogen oxide (NO) with hydroperoxyl radical $\left( \mathrm { H O } _ { 2 } \right)$ or organic peroxy radicals $( \mathsf { R O } _ { 2 } )$ – and loss (mainly reaction of $\Nu { 0 } _ { 2 }$ with OH). In the Dynamically Simple Model of Atmospheric Chemical Complexity (DSMACC) used in this study (see Sect. 2.3), the total reactive nitrogen $( \mathrm { N O } _ { y } )$ is constrained by either $\mathrm { N O } _ { 2 }$ or NO concentrations. We constrain the spatial distributions of the boundary layer $\mathrm { N O } _ { y }$ using satellite observations of $\Nu { 0 } _ { 2 }$ tropospheric vertical column density (VCD) of the QA4ECV (Quality Assurance for Essential Climate Variables) OMI $\mathrm { N O } _ { 2 }$ retrieval product (Boersma et al., 2018). Due to its short lifetime, $\mathrm { N O } _ { x }$ emitted from the surface mainly remains within the planetary boundary layer (PBL). Thus, satellite-retrieved VCDs are widely used in understanding the $\mathrm { N O } _ { 2 }$ distributions within the boundary layer instead of the whole troposphere (e.g., Cooper et al., 2020; Geddes et al., 2017).

We also constrain tropospheric CO, $\mathrm { C H } _ { 4 }$ , and $\mathrm { C H } _ { 2 } \mathrm { O }$ to better represent OH loss in the troposphere. Distributions of CO and $\mathrm { C H } _ { 2 } \mathrm { O }$ are taken from 4D variational data assimilation of the tropospheric CO column retrieved from the spaceborne MOPITT v7 instrument (Measurements Of Pollution In The Troposphere v7 TIR-NIR product; Deeter et al., 2017) and column $\mathrm { C H } _ { 2 } \mathrm { O }$ from the OMI version3 product (González Abad et al., 2015), respectively (Zheng et al., 2019). The $\mathrm { C H } _ { 4 }$ distributions are taken from data assimilation of surface $\mathrm { C H } _ { 4 }$ observations (Zhao et al., 2020a), mainly from the Earth System Research Laboratory of the US National Oceanic and Atmospheric Administration (NOAA/ESRL, Dlugokencky et al., 1994). The assimilated surface CO concentration and $\mathrm { C H } _ { 4 }$ profiles show good agreement with independent ground-based observations and aircraft observations, respectively (Zheng et al., 2019; Zhao et al., 2020a).

Meteorological conditions, mainly water vapor $( \mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g } ) } )$ and air temperature $( T _ { \mathrm { a } } )$ can also influence tropospheric OH. The $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ (represented as specific humidity) and $T _ { \mathrm { a } }$ are constrained by the second Modern-Era Retrospective analysis for Research and Applications (MERRA-2) reanalysis data from NASA’s Global Modeling and Assimilation Office (GMAO) (Gelaro et al., 2017).

# 2.2 The 3D atmospheric chemistry model simulations

The 3D distributions of monthly mean OH fields and OH precursors for the year 2010 are taken from the REF-C1 experiment of the IGAC/SPARC Chemistry–Climate Model Initiative Phase-1 (CCMI-1; Hegglin and Lamarque, 2015; Morgenstern et al., 2017). The REF-C1 experiment is driven by state-of-the-art historical forcings and sea surface temperatures from observations and covers 51 years (1960–2010).

We include simulations from two models with different chemical mechanisms: (1) the Community Earth System Model (CESM) using the Community Atmosphere Model version 4 as atmosphere component (CESM1–CAM4Chem; Tilmes et al., 2015, 2016) and (2) the GEOS-5 Chemistry Climate Model (GEOSCCM; Molod et al., 2012, 2015; Oman et al., 2011, 2013; Nielsen et al., 2017). The tropospheric chemistry of CESM1–CAM4Chem is based on MOZART-4 mechanisms with minor updates (Emmons et al., 2010; Lamarque et al., 2012) and the GEOSCCM is based on the Global Modeling Initiative (GMI) chemistry and transport model (Duncan et al., 2007), which was originally developed for the GEOS-Chem model. The CO, $\Nu { 0 } _ { 2 }$ , ${ { \mathrm O } _ { 3 } }$ , $\mathrm { C H } _ { 4 }$ , $\mathrm { C H } _ { 2 } \mathrm { O }$ mixing ratios, total column ${ { \mathrm O } _ { 3 } }$ , and meteorological conditions including $T _ { \mathrm { a } }$ and $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ simulated by CESM1–CAM4Chem and GEOSCCM in 2010 are compared with the observational data described in Sect. 2.1 in the Supplement (Figs. S1–S3). We chose the CESM1–CAM4Chem and GEOSCCM in this study since (1) their global mean OH concentrations and OH distributions (both horizontal and vertical) are around the multi-model mean values given by Zhao et al. (2019), albeit not at the extreme of the model distribution; (2) the two models include multiple primary non-methane volatile organic compounds (NMVOC) emissions (Morgenstern et al., 2017); and (3) the chemical box model DSMACC already includes the MOZART-4 and GEOS-Chem chemical mechanisms (Sect. 2.3), which are similar to that used in CESM1 CAM4-Chem and GEOSCCM, respectively.

A detailed description of the two model settings related to OH and the CCMI-1 model experiments can be found in Morgenstern et al. (2017) and Zhao et al. (2019).

# 2.3 The chemical box model DSMACC

Differing from 3D atmospheric chemistry models, which simulate the OH by gridded emissions inventories of its precursors, a chemical box model simulates OH by prescribing precursor concentrations and the meteorological conditions. Thus, one can estimate the sensitivity of OH to different precursor concentrations and meteorological parameters. Here, we use the Dynamically Simple Model of Atmospheric Chemical Complexity (DSMACC; Emmerson and Evans, 2009) to estimate the sensitivity of OH to chemical species including CO, $\Nu { 0 } _ { 2 }$ , ${ { \mathrm O } _ { 3 } }$ , $\mathrm { C H } _ { 4 }$ , $\mathrm { C H } _ { 2 } \mathrm { O }$ , the total column ${ { \mathrm O } _ { 3 } }$ , and meteorological conditions including $T _ { \mathrm { a } }$ and $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ , following the approach of Nicely et al. (2018).

The DSMACC model takes advantage of the kinetic preprocessor (KPP) to generate the FORTRAN code for a chosen chemical mechanism. In this study, the DSMACC model is compiled with MOZART-4 and GEOS-Chem chemical mechanisms, respectively, to be consistent with the associated 3D models CESM1–CAM4Chem and GEOSCCM. The clear-sky photolysis rates of chemical species are estimated by the tropospheric ultraviolet and visible (TUV) radiation model. Forced by meteorological variables $( \mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g } ) }$ , $T _ { \mathrm { a } }$ , and pressure), total column ${ { \mathrm O } _ { 3 } }$ , and gas concentrations simulated by the CESM1–CAM4Chem and GEOSCCM or generated from observations, and the diurnal cycle of the photolysis rates estimated by the TUV radiation model, the DSMACC is run forward until reaching the diurnal steady state of OH. Nicely et al. (2018) have estimated the response of OH to changes in OH precursors by conducting DSMACC model simulations for broad latitude and pressure bins. The results show that the $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ , $\mathrm { N O } _ { x }$ , total column ${ { \mathrm O } _ { 3 } }$ , and tropical expansion can lead to a positive trend in tropospheric OH, offsetting most of the negative trend led by the rising $\mathrm { C H } _ { 4 }$ concentrations from 1980 to 2015. Here, for each month, we run the DSMACC model for each model pixel of the 3D grid to better represent the heterogeneous spatial distributions of OH. For example, the CESM1–CAM4Chem has 144 (longitude) $\times 9 6$ (latitudes) $\times 1 3$ (pressure level) model grids in the troposphere. For each sensitivity experiment (Sect. 2.4), we therefore conduct 179 712 DSMACC model simulations (for the CESM1–CAM4Chem grid) each month.

\begin{array} { r l } & { [ \boldsymbol { \mathrm { O H } } ] _ { \mathrm { x k } _ { - } \mathrm { o b s } } = [ \boldsymbol { \mathrm { O H } } ] _ { \mathrm { m o d e l } } \times \frac { [ \boldsymbol { \mathrm { O H } } ] _ { \mathrm { D S M A C C } _ { - } \mathrm { x k } _ { - } \mathrm { o b s } } } { [ \boldsymbol { \mathrm { O H } } ] _ { \mathrm { D S M A C C } _ { - } \mathrm { R e f } _ { - } \mathrm { m o d e l } } } , } \\ & { \delta [ \boldsymbol { \mathrm { O H } } ] _ { x _ { k } } = [ \boldsymbol { \mathrm { O H } } ] _ { \mathrm { m o d e l } } - [ \boldsymbol { \mathrm { O H } } ] _ { \mathrm { x k } _ { - } \mathrm { o b s } } . } \end{array}
$$

# 2.5 Chemical loss of $\mathsf { C H } _ { 4 }$

We estimate the yearly tropospheric chemical loss of $\mathrm { C H } _ { 4 }$ through reaction with OH $( L _ { \mathrm { C H } _ { 4 } + \mathrm { O H } } )$ at global and regional scale from 2000 to 2009 by integrating the reaction of $\mathrm { C H } _ { 4 }$ with OH:

$$
L _ { \mathrm { C H _ { 4 } + O H } } = \sum _ { i } \sum _ { t } K \left( T _ { \mathrm { a } } \right) m \left( \mathrm { C H _ { 4 } } \right) [ \mathrm { O H } ] \delta t ,
$$

where $i$ is the index of the model pixel in the troposphere and $\delta t$ is the integration time step $( 3 \mathrm { h } )$ . The monthly 3D distributions of $\mathrm { C H } _ { 4 }$ mass $\mathrm { ( } m \mathrm { ( C H _ { 4 } ) } \mathrm { ) }$ ) during 2000–2009 are from data assimilation of surface $\mathrm { C H } _ { 4 }$ observations from NOAA/ESRL (Dlugokencky et al., 1994) and the $T _ { \mathrm { a } }$ distributions are from MERRA-2 reanalysis data (see Sect. 2.2). # 3 Results

# 3.1 Observation-based tropospheric OH

# 3.1.1 Global tropospheric OH burden

The global mean tropospheric column-averaged airmass-weighted OH $\left( [ \mathrm { O H } ] _ { \mathrm { t r o p - M } } \right)$ simulated by CESM1– CAM4Chem and GEOSCCM in 2010 are $1 1 . 9 \times 1 0 ^ { 5 }$ and $1 2 . 6 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ , respectively. By adjusting OH precursors and meteorological conditions (total column ${ { \mathrm O } _ { 3 } }$ , tropospheric ${ { \mathrm O } _ { 3 } }$ , CO, $\mathrm { C H } _ { 4 }$ , $\mathrm { C H } _ { 2 } \mathrm { O }$ , boundary layer $\Nu { 0 } _ { 2 }$ , $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ , and $T _ { \mathrm { a } , }$ ) to the observations using the DSMACC model, we estimated the observation-based $\mathrm { [ O H ] _ { t r o p - M } }$ to $\mathrm { b e 9 . 9 \times 1 0 ^ { 5 } }$ and $1 0 . 4 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ with CESM1–CAM4Chem and GEOSCCM chemical mechanisms, respectively (Fig. 1 and Table 2). Compared with the original OH fields simulated by CESM1– CAM4Chem and GEOSCCM, the observation-based OH fields reduce the model-simulated global $\mathrm { [ O H ] _ { t r o p - M } }$ by $\sim 2 \times 1 0 ^ { 5 } \mathrm { m o l e c . c m } ^ { - 3 }$ . The global $\mathrm { [ O H ] _ { t r o p - M } }$ estimated by the observation-based OH fields in this study is lower than the value estimated by Spivakovsky et al. (2000) $( 1 1 . 6 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 } ,$ ), which is used in the chemistry transport model (CTM) intercomparison experiment (TransCom-CH4) after being scaled by a factor of 0.92 (Patra et al., 2011) but consistent with those estimated by MCF-based inversions $( \sim 1 0 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ ; Bousquet et al., 2005; Krol and Lelieveld, 2003). The consistency with the MCF-based estimates indicates that our approach (correcting model bias through available observations) is capable of capturing the global OH burden.

Compared to the original $\mathrm { [ O H ] _ { t r o p - M } }$ simulated by CESM1–CAM4Chem and GEOSCCM, adjusting to observations reduces the $\mathrm { [ O H ] _ { t r o p - M } }$ by $2 { - } 8 \times \mathrm { \overline { { 1 0 ^ { 5 } } } m o l e c . } \mathrm { c m } ^ { - 3 }$ over most regions except the tropical forests. The Northern Hemisphere to Southern Hemisphere $\left( \mathrm { N } / \mathrm { S } \right)$ ratios of the simulated OH fields are reduced from 1.35 to 1.24 for CESM1–CAM4Chem and 1.26 to 1.15 for GEOSCCM. # 3.2.1 Contribution from CO

CO is the largest OH sink in the troposphere (Lelieveld et al., 2016; Zhao et al., 2020b). The sensitivity simulation CO_obs shows that a 1 ppbv increase in CO can result in a decrease in OH by up to more than $3 \times 1 0 ^ { 4 } \mathrm { m o l e c . c m } ^ { - 3 }$ (Fig. S7). Compared with the CO distributions from inversions that assimilated MOPITT observations, the CESM1– CAM4Chem underestimates the global tropospheric mean CO mixing ratio by 24 ppbv (Fig. S1). Based on the DS-MACC simulations CO_obs and Ref_CESM (Table 1), we find that the negative bias in CO contributes most to the difference in the modeled versus observation-based $\mathrm { [ O H ] _ { t r o p - M } }$ $( 1 . 3 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ ; Table 4 and Fig. 3). The underestimation of CO is common in atmospheric models and was treated either as a cause or an effect of the overestimated OH in previous studies (Naik et al., 2013; Monks et al., 2015; Strode et al., 2015). For example, based on the Atmospheric Chemistry and Climate Model Intercomparison Project (AC-CMIP) simulations, Naik et al. (2013) found that the positive bias in OH was most likely due to the underestimation of CO when compared with both satellite and surface observations. In contrast, Strode et al. (2015) did sensitivity simulations using the GEOSCCM model and showed that reducing OH bias could improve the accuracy of modeled CO. In this study, we do not intend to solve the cause-and-effect issue between CO and OH since the discrepancy in $\mathrm { [ O H ] _ { t r o p - M } }$ of $1 . 3 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ could also be understood as the global tropospheric [OH] changes that would be needed to simulate the observed CO.

# 3.2.2 Contribution from tropospheric ${ { \sf O } _ { 3 } }$

Tropospheric ${ { \mathrm O } _ { 3 } }$ can contribute to both primary and secondary OH production. Compared to satellite observations from OMI, CESM1–CAM4Chem simulations show a large overestimation of tropospheric ${ { \mathrm O } _ { 3 } }$ over the $1 5 \mathrm { - } 6 0 ^ { \circ } \mathrm { N }$ region (up to 14 DU, $40 \%$ ) and an underestimation (14 DU, $40 \%$ over the tropics and Southern Hemisphere (Fig. S1).

At the global scale, the model bias in ${ { \mathrm O } _ { 3 } }$ leads to a negative bias on $\mathrm { [ O H ] _ { t r o p - M } }$ by $0 . 3 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ , much smaller than that caused by CO (Table 4). However, at the regional scale, The CESM1–CAM4Chem simulated $\mathrm { [ O H ] _ { t r o p - M } }$ is enhanced by $\sim 1 \times 1 0 ^ { 5 } \mathrm { m o l e c . c m } ^ { - 3 }$ over the tropics $( 1 5 ^ { \circ } S -$ $1 5 ^ { \circ } \mathrm { N } )$ and $\sim 0 . 5 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ over the mid-Southern Hemisphere $( 1 5 { - } 6 0 ^ { \circ } \ : \mathrm { S } )$ , while it is reduced by $0 . 1 { - } 0 . 3 \times$ $1 0 ^ { 5 } \mathrm { m o l e c . c m } ^ { - 3 }$ over the mid-Northern Hemisphere ( $1 5 -$ $6 0 ^ { \circ } \mathrm { N } )$ when adjusted to OMI/MLS tropospheric column ${ { \mathrm O } _ { 3 } }$ (Fig. 2). The adjustment reduces the $\mathrm { N } / \mathrm { S }$ ratio of $\mathrm { [ O H ] _ { t r o p - M } }$ by 0.07; however, it still cannot explain the overestimation of the $\mathrm { N } / \mathrm { S }$ ratio but leads to a larger correction than the one with CO alone.

# 3.2.3 Contribution from boundary layer ${ \mathsf { N O } } _ { 2 }$

The sensitivity of OH to $\Nu { 0 } _ { 2 }$ is highly variable. We estimate that a 1 ppbv increase in $\Nu { 0 } _ { 2 }$ can lead to a change of OH ranging from $- 3 \times 1 0 ^ { 6 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ to more than $+ 1 0 \times 1 0 ^ { 6 }$ molec. $\mathrm { c m } ^ { - 3 }$ , depending on the mixing ratio of NMVOCs (represented as ${ \mathrm { H C H O + } }$ isoprene) and $\Nu { 0 } _ { 2 }$ (Fig. S7). Compared to the QA4ECV $\Nu { 0 } _ { 2 }$ retrieval product, CESM1–CAM4Chem overestimates tropospheric $\Nu { 0 } _ { 2 }$ over most regions except northern China and tropical forests (Fig. S1).

At the global scale, the overestimation of $\Nu { 0 } _ { 2 }$ leads to a positive bias in $\mathrm { [ O H ] _ { t r o p - M } }$ by $0 . 3 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ (Table 4). At the regional scale, correcting for the PBL $\mathrm { N O } _ { 2 }$ does not influence the $\mathrm { N } / \mathrm { S }$ ratio of OH. As shown in Fig. 3, the overestimation of PBL $\mathrm { N O } _ { 2 }$ results in a positive bias in $\mathrm { [ O H ] _ { t r o p - M } }$ over most of the continental regions. Over tropical and temperate oceans, one can also see that the slight overestimation in $\mathrm { N O } _ { 2 }$ leads to a significate positive bias in OH by $0 . 5 \mathrm { - } 1 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ since the sensitivity of $\mathrm { \ [ O H ] _ { t r o p - M } }$ to $\mathrm { N O } _ { 2 }$ can be very high ( $1 0 ^ { 7 }$ molec. $\mathrm { c m } ^ { - 3 } \mathrm { p p b v } ^ { - 1 } \ \mathrm { N O } _ { 2 } )$ over the regions with low $\mathrm { N O } _ { x }$ and NMVOC mixing ratios. Over northern China, although the model shows a large underestimation in $\Nu { 0 } _ { 2 }$ (Fig. S1), the $\mathrm { [ O H ] _ { t r o p - M } }$ is slightly smaller after adjustment. This is because the OH is not sensitive to an increase in $\Nu { 0 } _ { 2 }$ over high $\Nu { 0 } _ { 2 }$ regions or even shows a negative response (Fig. S7).

# 3.2.4 Contribution from total column ${ { \mathrm O } _ { 3 } }$

Total column ${ { \mathrm O } _ { 3 } }$ mainly influences $\mathrm { O } ^ { 1 } ( \mathrm { D } )$ photolysis through absorbing UV radiation. The CESM1–CAM4Chem simulation mainly underestimates the total ${ { \mathrm O } _ { 3 } }$ columns by up to $\sim 1 0 \mathrm { D U }$ over tropical regions compared with the SBUV MOD observations (Fig. S2). On a global scale, the underestimation of the total column ${ \bf O } _ { 3 }$ can lead to an overestimation of the $\mathrm { [ O H ] _ { t r o p - M } }$ by $0 . 4 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ (Table 4 and Fig. 3), comparable with that due to tropospheric ${ { \mathrm O } _ { 3 } }$ .

# 3.2.5 Contribution from $\mathsf { C H } _ { 4 }$ and ${ \sf C H } _ { 2 } { \sf O }$

In CCMI-1 simulations, atmospheric chemistry models prescribe the lower boundary conditions for $\mathrm { C H } _ { 4 }$ following the Representative Concentration Pathway (RCP6.0). Compared to the posterior $\mathrm { C H } _ { 4 }$ fields from inversions by assimilating the surface $\mathrm { C H } _ { 4 }$ observations, the tropospheric mean $\mathrm { C H } _ { 4 }$ mixing ratios used in the CESM–CAM4Chem are $\sim 8 0$ ppbv lower over the tropical and extratropical regions with high biomass burning and anthropogenic emissions and $\sim 4 0$ ppbv lower over other regions. However, due to the low sensitivity of [OH] to $\mathrm { C H } _ { 4 }$ changes (Fig. S1), the underestimation in $\mathrm { C H } _ { 4 }$ only leads to a small positive bias in the global mean $\mathrm { [ O H ] _ { t r o p - M } }$ by $0 . 1 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ .

The CESM1–CAM4Chem overestimates $\mathrm { C H } _ { 2 } \mathrm { O }$ by more than $50 \%$ over land but slightly underestimates $\mathrm { C H } _ { 2 } \mathrm { O }$ over tropical oceans (Fig. S1). Since $\mathrm { C H } _ { 2 } \mathrm { O }$ contributes to only a small part $(6 \% )$ of the total OH loss (Zhao et al., 2020b), the large bias in the CESM1–CAM4Chem simulated $\mathrm { C H } _ { 2 } \mathrm { O }$ only leads to a small positive bias global mean $\mathrm { [ O H ] _ { t r o p - M } }$ by $0 . 1 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ (Fig. 3).

# 3.2.6 Contribution from meteorological conditions

$\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ is a major OH precursor that contributes to the primary production of $\mathrm { O H }$ and $T _ { \mathrm { a } }$ can influence OH production and loss rates. Compared to MERRA-2 reanalysis data, CESM1–CAM4Chem overestimates zonally averaged $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ mixing ratios near the surface and around $8 0 0 \mathrm { h P a }$ by $\sim 1 . 5 \mathrm { g } \mathrm { k g } ^ { - 1 }$ (Fig. S3). The sensitivity experiments show that a change in specific humidity by $1 \mathrm { g } \mathrm { \bar { k } g } ^ { - 1 }$ can lead to a change in OH by $> 3 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ over the regions with high $\mathrm { O } ( ^ { 1 } \mathrm { D } )$ photolysis and low NMVOC mixing ratios (Fig. S7). As shown in Table 4 and Fig. 3, globally, the model bias in $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ only leads to a small bias $( 0 . { \dot { 1 } } \times 1 0 ^ { 5 } { \mathrm { m o l e c . } } { \mathrm { c m } } ^ { - 3 } )$ ) $\mathrm { [ O H ] _ { t r o p - M } }$ , but regionally, the model bias in $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ can lead to a bias in $\mathrm { [ O H ] _ { t r o p - M } }$ by the magnitude of $5 \times 1 0 ^ { 5 } \mathrm { m o l e c . c m } ^ { - 3 }$ , even larger than that induced by the bias in CO. For $T _ { \mathrm { a } }$ , the model only shows a small bias $( < 1 ^ { \circ } \mathrm { C } )$ compared with MERRA-2 reanalysis data (Fig. S3). Thus, model bias in $\mathrm { [ O H ] _ { t r o p - M } }$ induced by $T _ { \mathrm { a } }$ is negligible (Fig. 3).

# 3.3 Chemical sinks of $\mathsf { C H } _ { 4 }$ as estimated by observation-based OH fields

# 3.3.1 Global and regional OH chemical sink of $\mathsf { C H } _ { 4 }$

Using the observation-based OH field, we estimate that the global tropospheric $\mathrm { C H } _ { 4 }$ loss by reaction with tropospheric OH $( L _ { \mathrm { C H } _ { 4 } + \mathrm { O H } } )$ averaged during the period 2000 through 2009 is 434 and $4 6 1 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ for CESM1– CAM4Chem and GEOSCCM, respectively. These estimates are about $1 0 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ lower than estimated by the original model-simulated OH fields (540 and $5 6 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ , respectively; Table 2). The corresponding $\mathrm { C H } _ { 4 }$ lifetimes against tropospheric OH loss estimated by the two observation-based OH fields are 11.4 and 10.7 year for CESM1–CAM4Chem and GEOSCCM, respectively, well within the range estimated by Prather et al. (2012) based on the MCF-inversions $( 1 1 . 2 \pm 1 . 3 \mathrm { y r } )$ and much longer than estimated by the original model-simulated OH fields (9.1 years for CESM1– CAM4Chem and 8.7 years for GEOSCCM).

As shown in Table 3, more than $70 \%$ of the tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ occurs over tropical regions mainly due to both high OH and $T _ { \mathrm { a } }$ . Constraining the tropospheric OH by precursor concentrations reduces the tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ by $\sim 3 0 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $( 1 6 \% )$ over the southern tropics, $\sim 5 0 \mathrm { \dot { T } g y r ^ { - 1 } }$ $(21 \% )$ over the northern tropics, and $\sim 2 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $( 2 5 \% )$ over the northern middle to high latitude as estimated by both CESM1–CAM4Chem and GEOSCCM OH fields. Over the southern middle- to high-latitude regions, there are only limited changes $( \sim 5 \mathrm { T g } \mathrm { y r } ^ { - 1 } ,$ ) in tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ Thus, constraining tropospheric OH by precursor concentrations changes the inter-hemispheric distribution of $L _ { \mathrm { C H _ { 4 } + O H } }$ The values of $L _ { \mathrm { C H _ { 4 } + O H } }$ estimated by the observationbased OH fields are $\sim 3 5$ and $\sim 7 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ lower than that estimated by the corresponding original model-simulated OH fields over the Southern and Northern Hemisphere, respectively (Table 3). Thus, the inter-hemispheric difference of $L _ { \mathrm { C H _ { 4 } + O H } }$ (north–south) estimated by observationbased OH fields $( 6 0 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ by CESM1–CAM4Chem and $4 8 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ by GEOSCCM) is $\sim 4 0 \%$ lower than estimated by the original model-simulated OH fields $( 9 8 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ by CESM1–CAM4Chem and $8 1 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ by GEOSCCM).

# 3.3.2 Global total chemical sink of $\mathsf { C H } _ { 4 }$

We estimate the global total $\mathrm { C H } _ { 4 }$ chemical sink for 2000– 2009 by gathering (1) the tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ estimated using the original model-simulated and observation-based OH fields, (2) the $\mathrm { C H } _ { 4 }$ loss in the stratosphere $( 2 6 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ estimated by CESM1 CAM4-chem and $3 6 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ estimated by GEOSCCM simulations) and $\mathrm { C H } _ { 4 }$ oxidized by chlorine $( 1 1 \mathrm { T g } \mathrm { y r } ^ { - 1 } )$ given by Saunois et al. (2020). We then compare the chemical sink estimated in this study with that estimated by the bottom-up and top-down methods given by the previous GCP global $\mathrm { C H } _ { 4 }$ budget (Saunois et al., 2016, 2020).

As shown in Fig. 4, the bottom-up estimates in the GCP global $\mathrm { C H } _ { 4 }$ budget (blue bars) have a large range $( 4 8 3 -$ $7 3 8 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ in Saunois et al., 2016 and $4 8 9 { - } 7 4 9 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ in Saunois et al., 2020), much higher than those from the top-down method $( 5 1 4 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ in Saunois et al., 2016 and $4 5 9 { - } 5 1 6 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ in Saunois et al., 2020). The $\mathrm { C H } _ { 4 }$ sinks simulated by CESM1–CAM4Chem $( 5 4 9 \mathrm { T g } \mathrm { y r } ^ { - 1 } ) $ ) and GEOSCCM $( 5 8 5 \mathrm { T g } \mathrm { y r } ^ { - 1 } )$ were included in the bottomup estimates in Saunois et al. (2020) (green bar) and are slightly lower than the average value estimated using different OH fields $( 5 9 5 \mathrm { T g } \mathrm { y r } ^ { - 1 } $ ).

Constraining the OH field with available precursor observations to correct the global OH, the magnitude of the methane loss is more in line with top-down methane inversions. Therefore, we partly reconcile the bottom-up and top-down estimates of the $\mathrm { C H } _ { 4 }$ sink. Although only two of seven bottom-up models synthesized in Saunois et al. (2020) are considered in this study, our approach can be generalized to other chemistry–climate models. Instead of directly using the OH fields simulated from an atmospheric chemistry model, the bottom-up estimates can use the precursor observations and box-model based approach proposed here to reduce model biases of OH fields.

# 3.3.3 Contribution from the model biases of individual OH precursors to chemical sink of $\mathsf { C H } _ { 4 }$

We further quantify the influence of model biases in individual OH precursors on the bottom-up estimates of $\mathrm { C H } _ { 4 }$ chemical sink $( \delta L _ { \mathrm { C H _ { 4 } + O H _ { \mathrm { - } } X k } } )$ . At the global scale, the underestimation of CO and total column ${ { \mathrm O } _ { 3 } }$ and the overestimation of $\Nu { 0 } _ { 2 }$ by the CESM1–CAM4Chem lead to a positive bias of $6 0 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $( 1 1 \% )$ , $2 2 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $(4 \% )$ , and $2 2 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $(4 \% )$ in tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ (Fig. 4 and Table 4), respectively, while an underestimation of tropospheric ${ { \mathrm O } _ { 3 } }$ leads to a negative bias of $1 7 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $( 3 \% )$ in tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ . Although the model bias of $\mathrm { [ O H ] _ { t r o p - M } }$ induced by $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ is negligible on the global scale, the observation-based adjustment of $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ leads to a reduction in tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ by $1 0 \mathrm { { T g } y r ^ { - 1 } } \ ( 2 \% )$ , since the model overestimation of $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ is concentrated over the middle- to low-latitude regions where tropospheric $\mathrm { C H } _ { 4 }$ oxidation mainly occurs (Fig. S3). The model bias in $\mathrm { C H } _ { 2 } \mathrm { O }$ and $\mathrm { C H } _ { 4 }$ itself leads to a small positive bias of $\sim 1 \%$ respectively on $L _ { \mathrm { C H _ { 4 } + O H } }$ .

As the tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ mainly occurs over the middle- to low-latitude regions, the biases in $\mathrm { [ O H ] _ { t r o p - M } }$ over the high latitudes (north of $6 0 ^ { \circ } \mathrm { N }$ or south of $6 0 ^ { \circ } S$ ) due to an overestimation of CO and underestimation of $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ do not substantively contribute to the bias in $L _ { \mathrm { C H _ { 4 } + O H } }$ (Fig. 2). Over the regions north of $1 5 ^ { \circ } \mathrm { N }$ , nearly all the precursors considered in this study contribute to the overestimation of $L _ { \mathrm { C H _ { 4 } + O H } }$ $5 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ in total), of which $47 \%$ $( 2 6 \mathrm { T g } \mathrm { y r } ^ { - 1 } )$ is contributed by model underestimation of CO. South of $1 5 ^ { \circ } \mathrm { N }$ , the underestimation of tropospheric ${ { \mathrm O } _ { 3 } }$ results in an underestimation of $L _ { \mathrm { C H _ { 4 } + O H } }$ by $2 2 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ partly offsetting the overestimation of $L _ { \mathrm { C H _ { 4 } + O H } }$ induced by CO $( 3 4 \mathrm { T g } \mathrm { y r } ^ { - 1 } )$ ) and other precursors $( 4 0 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ in total) (Fig. 2). As previously mentioned, the inter-hemispheric difference of $L _ { \mathrm { C H _ { 4 } + O H } }$ derived from the observation-based OH fields is $4 8 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ smaller than estimated using the OH field originally simulated by CESM1–CAM4Chem. The biases in CO, tropospheric ${ \bf O } _ { 3 }$ , and boundary layer $\Nu { 0 } _ { 2 }$ , lead to an overestimation of the inter-hemispheric difference of tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ by 15, 15, and $9 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ , respectively, dominating the bias in the inter-hemispheric difference in tropospheric $L _ { \mathrm { C H } _ { 4 } + \mathrm { O H } }$ .

# 4 Conclusions and discussion

In this study, we aim to reconcile the top-down and bottomup estimates of the major global $\mathrm { C H } _ { 4 }$ sink and to quantify the contribution of each factor to the overestimation of tropospheric OH that is generally found in atmospheric chemistry models and to the consequent overestimation of $\mathrm { C H } _ { 4 }$ chemical loss in the bottom-up studies. To do so, we propose a new approach based on precursor observations and a chemical box model to improve the 3D distributions of tropospheric OH radicals issued from atmospheric chemistry models.

# 4.1 Conclusions for OH

We estimate two 3D observation-based OH fields based on three components: (i) simulated tropospheric OH and related chemical species from global 3D atmospheric chemistry models (here CESM1–CAM4Chem and GEOSCCM), (ii) sensitivities of tropospheric OH to its precursors in each model grid cell estimated by the chemical box model DS-MACC using a chemical mechanism similar to the 3D model, and (iii) observations of chemical species related to OH production and loss (CO, ${ { \mathrm O } _ { 3 } }$ , boundary layer $\mathrm { N O } _ { 2 }$ , $\mathrm { C H } _ { 4 }$ , $\mathrm { C H } _ { 2 } \mathrm { O }$ , and total column ${ { \mathrm O } _ { 3 } }$ ) and meteorological conditions $( \mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ and $T _ { \mathrm { a } }$ ). The chemical box model DSMACC can be compiled using different chemical mechanisms, making it possible to apply this approach to other atmospheric chemistry models and improve their representation of OH.

The global $\mathrm { [ O H ] _ { t r o p - M } }$ estimated from observationbased OH fields is $\sim 1 \dot { 0 } \times 1 0 ^ { 5 } \mathrm { m o l e c . c m } ^ { - 3 }$ in 2010 based on two different chemical mechanisms, which is $2 \times$ $1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ lower than the original model-simulated global $\mathrm { [ O H ] _ { t r o p - M } }$ , consequently reaching consistency with the value derived by MCF-based inversions (around $1 0 \times$ $1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ ; Bousquet et al., 2005; Krol and Lelieveld, 2003). The observation-based adjustments also change the latitudinal distribution of OH, reducing its north–south ratios from 1.35 and 1.26 to 1.24 and 1.15 for CESM1– CAM4Chem and GEOSCCM, respectively, closer to, albeit not as low as, the one obtained from MCF-based inversions (slightly smaller than 1).

Based on the simulations from CESM1–CAM4Chem, globally, the overestimation of $\mathrm { [ O H ] _ { t r o p - M } }$ arises mainly from the underestimation of CO and total column ${ \bf O } _ { 3 }$ , and the overestimation of boundary layer $\Nu { 0 } _ { 2 }$ , which contribute $1 . 3 \times$ $1 0 ^ { 5 }$ , $0 . 4 \times 1 0 ^ { 5 }$ , and $0 . 3 \times 1 0 ^ { \bar { 5 } }$ molec. $\mathrm { c m } ^ { - 3 }$ , respectively, to the bias in $\mathrm { [ O H ] _ { t r o p - M } }$ . For the $\mathrm { N } / \mathrm { S }$ ratio of $\mathrm { [ O H ] _ { t r o p - M } }$ , the positive bias in $\mathrm { [ O H ] _ { t r o p - M } }$ over the Northern Hemisphere $( 0 . 1 -$ $0 . 3 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 } ,$ ) and the negative bias over the tropics and Southern Hemisphere $( 0 . { \bar { 5 } } - 1 . 0 \times 1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - \bar { 3 } }$ ) due to tropospheric ${ { \mathrm O } _ { 3 } }$ dominate the higher $\mathrm { N } / \mathrm { S }$ ratio of $\mathrm { [ O H ] _ { t r o p - M } }$ estimated by the CESM1–CAM4Chem than the observation-based OH field. At the regional scale, the model bias in $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ can lead to bias in $\mathrm { [ O H ] _ { t r o p - M } }$ even larger than that induced by CO.

# 4.2 Conclusions for the $\mathsf { C H } _ { 4 }$ sink

The global $\mathrm { C H } _ { 4 }$ loss by reaction with tropospheric OH $( L _ { \mathrm { C H } _ { 4 } + \mathrm { O H } } )$ estimated from the observation-based OH fields is 434 and $4 6 1 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ for CESM1–CAM4Chem and GEOSCCM, respectively, averaged over 2000 to 2009, which is lower than that estimated from the original modelsimulated OH fields by around $1 0 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ . Based on the results from CESM1–CAM4Chem, at the global scale, the underestimation of CO and total column ${ { \mathrm O } _ { 3 } }$ , and overestimation of $\Nu { 0 } _ { 2 }$ lead to positive biases in tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ by $6 0 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $( 1 1 \% )$ , $2 2 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $(4 \% )$ , and $2 2 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ $(4 \% )$ , respectively, while an underestimation of tropospheric ${ { \mathrm O } _ { 3 } }$ leads to a negative bias in tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ by $1 7 \mathrm { T g } \mathrm { y r } ^ { - 1 } ( 3 \% )$ . The inter-hemispheric difference in the tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ is therefore reduced by $40 \%$ (around $3 5 \mathrm { T g } \mathrm { y r } ^ { - 1 } .$ ) when estimated using the observation-based OH field. Although the bias in the $\mathrm { N } / \mathrm { S }$ ratio of $\mathrm { [ O H ] _ { t r o p - M } }$ is dominated by the tropospheric ${ { \mathrm O } _ { 3 } }$ concentration, the positive bias in the inter-hemispheric difference of $L _ { \mathrm { C H _ { 4 } + O H } }$ is determined together by the biases in CO $( 1 5 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ ), tropospheric ${ { \mathrm O } _ { 3 } }$ $( 1 5 \mathrm { T g } \mathrm { y r } ^ { - 1 } )$ , and boundary layer $\Nu { 0 } _ { 2 }$ $( 9 \mathrm { T g } \mathrm { y r } ^ { - 1 } )$ .

Using the tropospheric $L _ { \mathrm { C H _ { 4 } + O H } }$ estimated with our observation-based OH fields, the global total $\mathrm { C H } _ { 4 }$ chemical sink is $4 7 1 { - } 5 0 8 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ . This quantification is more consistent with top-down estimates in the previous GCP global $\mathrm { C H } _ { 4 }$ budget $( 4 5 9 - 5 1 6 \mathrm { T g } \mathrm { y r } ^ { - 1 }$ , Saunois et al., 2016, 2020) than it was before the adjustment $( 5 7 7 - 6 1 2 \mathrm { T g } \mathrm { y r } ^ { - 1 } ,$ ). The bottom-up method in the previous GCP global $\mathrm { C H } _ { 4 }$ budget estimated the $\mathrm { C H } _ { 4 }$ chemical sink directly using the OH fields simulated by atmospheric chemistry models. However, the uncertainties in the model-simulated OH lead to an unreliable range in the bottom-up estimated $\mathrm { C H } _ { 4 }$ chemical sink, much higher than that estimated by the top-down method. Our results highlight that constraining the OH fields using available precursor observations can improve the bottom-up estimates of the $\mathrm { C H } _ { 4 }$ sink and help reconcile the difference between the top-down and bottom-up estimates of the $\mathrm { C H } _ { 4 }$ sink.

# 4.3 Discussion

Although the observation-based 3D OH fields presented in this study can capture the global tropospheric OH burden and chemical loss of $\mathrm { C H } _ { 4 }$ , unresolved uncertainties and limitations remain.

1. The method presented in this study cannot improve the chemical mechanisms in the models and does not fully explain the overestimation of the $\mathrm { N } / \mathrm { S }$ ratios of OH.

The differences in global $\mathrm { [ O H ] _ { t r o p - M } }$ between the two observation-based OH fields estimated from CESM1– CAM4Chem and GEOSCCM simulations is $0 . 5 \times$ $1 0 ^ { 5 }$ molec. $\mathrm { c m } ^ { - 3 }$ . Besides precursor concentrations, the inter-model difference in tropospheric OH is partly attributable to their differences in chemical mechanisms (Nicely et al., 2018, 2020). As discussed by Murray et al. (2021), the oxidative efficiency of NMVOCs and lifetime of $\mathrm { N O } _ { x }$ simulated by different models can largely determine inter-model differences in tropospheric OH and their responses to changes in precursors. Reducing the uncertainties due to the modeled chemical mechanisms relies on additional observations to improve the simulation of the oxidative efficiency of NMVOCs and $\mathrm { N O } _ { x }$ lifetime, which is beyond the scope of our study.

The $\mathrm { N } / \mathrm { S }$ ratio of $\mathrm { [ O H ] _ { t r o p - M } }$ after observation-based adjustment is still higher than the one obtained from MCF-inversions (less than 1.0). This difference indicates that the overestimation of the $\mathrm { N } / \mathrm { S }$ ratio by atmospheric models cannot be fully explained by the underestimation of CO and overestimation of ${ { \mathrm O } _ { 3 } }$ over the Northern Hemisphere as mentioned in previous studies (Naik et al., 2013). The overestimation of the $\mathrm { N } / \mathrm { S }$ ratio may also be attributable to chemical mechanisms included in the atmospheric chemistry models. Both CESM1–CAM4Cchem and GEOSCCM do not include the OH recycling by isoprene and simulate low OH values in regions with high NMVOC emissions, such as rain forests in the Southern Hemisphere (Zhao et al., 2019). Including the chemical mechanism such as OH recycling by isoprene (Lelieveld et al., 2008) would help to further reduce the N/S ratio for model-simulated OH fields.

2. The constraints brought on tropospheric OH are limited by quality of observations and time resolution of available model outputs.

Data constraining the OH precursors come mainly from satellite observations and reanalysis data, the uncertainties of which are not considered in this study. For example, the MERRA-2 reanalysis data significantly overestimate $\mathrm { H } _ { 2 } \mathrm { O } _ { ( \mathrm { g ) } }$ in the upper troposphere (Jiang et al., 2015); The QA4ECV tropospheric $\Nu { 0 } _ { 2 }$ VCD is lower compared with surface observations under the extremely high-pollution case compared with surface observations (Compernolle et al., 2020). The performance of this method depends on the accuracy of observations used to constrain individual factors. Data products regularly improve and, since the sensitivity of OH to each precursor is estimated by the chemical box model, we can easily improve the OH using the updated observational datasets.

The OMI measures concentrations of chemical species around local time $1 3 { : } 3 0 \mathrm { L T }$ , but most of the CCMI models only provide monthly means for 3D distribution of chemical concentrations. The monthly mean $\Nu { 0 } _ { 2 }$ and ${ { \mathrm O } _ { 3 } }$ concentrations simulated by 3D models are therefore constrained only by such afternoon observations. For ${ { \mathrm O } _ { 3 } }$ , of which the tropospheric mean lifetime is $2 3 . 4 \pm 2 . 2 \mathrm { d }$ (Young et al., 2013), we assume that not considering diurnal variations only has a small influence. This is not the case for $\Nu { 0 } _ { 2 }$ with a much shorter lifetime $( \sim 1 \mathrm { d }$ , Jaffe et al., 2003). By comparing the tropospheric $\mathrm { N O } _ { 2 }$ VCDs observed by SCIAMACHY (SCanning Imaging Absorption SpectroMeter for Atmospheric Chartography; overpass time around local time 10:00 LT) with OMI, previous studies show that the tropospheric $\Nu { 0 } _ { 2 }$ VCDs have significant diurnal variations (Boersma et al., 2008, 2009). Diurnal variations of $\Nu { 0 } _ { 2 }$ VCDs are controlled by complex factors including local emissions, photochemistry, deposition, advection, etc., and vary among different seasons over different regions (Boersma et al., 2008, 2009). Considering the diurnal cycle of $\mathrm { N O } _ { 2 }$ photolysis, tropospheric $\mathrm { N O } _ { 2 }$ VCDs over remote regions should be lower during daytime than nighttime (Cheng et al., 2019). Constraining the model-simulated monthly mean $\Nu { 0 } _ { 2 }$ VCDs with satellite data at the overpass time leads to an overestimation of the high bias of modeled tropospheric $\Nu { 0 } _ { 2 }$ VCDs. Thus, the $0 . 3 \times 1 0 ^ { 5 } \mathrm { m o l e c . } \mathrm { c m } ^ { - 3 }$ estimated in this study gives an upper limit of the high bias in global $\mathrm { [ O H ] _ { t r o p - M } }$ due to boundary layer $\Nu { 0 } _ { 2 }$ .

Since we only have the tropospheric $\mathrm { N O } _ { 2 }$ VCDs, another key factor that could influence the tropospheric OH burden but is unconstrained in this study is $\Nu { 0 } _ { 2 }$ in the free troposphere. Although the $\Nu { 0 } _ { 2 }$ mixing ratio is usually less than 1 ppbv in the free troposphere, the sensitivity of OH to $\Nu { 0 } _ { 2 }$ can be very high in low $\mathrm { N O } _ { 2 }$ regions. However, a potential model bias due to lightning $\mathrm { N O } _ { x }$ emissions, which had proven to contribute significantly to the upper-tropospheric OH burden (Murray et al., 2013; Turner et al., 2018), is not adjusted in our study. Satellite retrievals for upper-tropospheric $\Nu { 0 } _ { 2 }$ (e.g., Belmonte Rivas et al., 2015; Marais et al., 2021) could help quantify OH biases due to free tropospheric $\Nu { 0 } _ { 2 }$ and the contribution of lightning $\mathrm { N O } _ { x }$ emissions.

# 4.4 Future developments

The new approach proposed here to improve the 3D OH fields and chemical loss of $\mathrm { C H } _ { 4 }$ can be applied broadly. It relies on observations of OH precursor concentrations that can be applied efficiently to any atmospheric chemistry model with a box model (0D) available. Here, we only apply this method to two models for 1 year (2010), and both of them agree with MCF-based inversions in terms of the global OH burden. One future research development is to generate observation-based OH fields for all the atmospheric chemistry models included in the GCP global $\mathrm { C H } _ { 4 }$ budget and over a longer time period, especially for the models that simulate extremely high or low OH. This will allow us to see whether our results can be generalized with a larger range of OH and $\mathrm { C H } _ { 4 }$ losses and to see if a higher consistency can also be achieved on longer timescales. It will also be important to assess how much uncertainty in OH means and trends can be further reduced and achieved in detail.

The $\mathrm { C H } _ { 4 }$ emissions from top-down approaches mostly used a single OH field from Spivakosky et al. (2000), which relies on climatological data without any interannual variations. Some $\mathrm { C H } _ { 4 }$ inversions used the OH fields from chemistry–climate or chemistry transport models with the known aforementioned biases that may lead to bias in the inverted surface $\mathrm { C H } _ { 4 }$ fluxes. Our OH product could be used instead in $\mathrm { C H } _ { 4 }$ inversions to better infer $\mathrm { C H } _ { 4 }$ emissions and reduce the uncertainties in the global methane budget. Each modeling group could also generate their own corrected OH for the purpose of methane atmospheric inversions. Further work is necessary to consider the interannual changes in our observation-based estimates.
